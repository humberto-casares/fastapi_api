# FastAPI Users API

Simple FastAPI microservice example with async CRUD operations for a `users` table using `asyncpg`, custom DB helpers, and yoyo migrations.

- Swagger UI: [http://localhost:8089/management/swagger-ui](http://localhost:8089/management/swagger-ui)
- OpenAPI JSON: [http://localhost:8089/management/openapi.json](http://localhost:8089/management/openapi.json)
